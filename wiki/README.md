# Wiki Mirror

These pages mirror the project wiki so the documentation is available in the source repository and on GitHub even before the GitHub wiki git endpoint is materialized.

Once GitHub creates `allofitnow/houdini-aws-portal.wiki.git`, push the prepared wiki clone with:

```bash
git -C /tmp/houdini-aws-portal-wiki push github master
```
